# Grocery List Manager (MiniProject 3)

###
Ivy Bowers

This website is a grocery list manager made with React, keeping the frontend (UI) simple and user-friendly. It's okay of the items only contain text-based information (image is not a must). All componenets should have proper styles.

### Major Sections in the page:
- Component-based implementation
- Grocery Item Component
- Buttons/labels/etc. so a user can add/update/delete items
- Successful application of `map`, `filter`, `reduce` (at least one, based on necessity)
---

### Functions of the grocery list:
- Add/insert new grocery items to the list
- Update/modify existing item
- Delete/remove some existing items from the list
- Display/view the current items (as an ul/li)
